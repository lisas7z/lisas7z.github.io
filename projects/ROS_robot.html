Project Background

ENPH 353 is a guided, self-directed project course offered by UBC Engineering Physics. In this course, the goal is to practice techniques and developments in control software and data classification, and the learning will be demonstrated through a final competition.

In the 2024 competition, our task is to develop an autonomous robot in the Gazebo simulation environment that is capable of terrain-navigation (road, grass, and mountain), barrier detection (pedestrain and other moving obstacles), and character-recognition (clueboard reading). The scoring is based on correctly predicting clue values and adhering to traffic rules within a four-minute simulation time.

High-Level Summary

This project is a collaboration between Andrew Du and Lisa Shi, by incorporating machine learning, control algorithm, and computer vision techniques, we achieved:

  1. navigating through a variety of terrain using masking and filtering algorithm, along with PID control for line following;
  2. avoiding moving barriers through motion detection;
  3. accurately capturing characters on the clueboards through thresholding, filtering, and prespective transformation;
  4. generating a large dataset of synthetic characters using image transformations to train a robust CNN classifier;
  5. developing a ROS-based architecture using ROS nodes and topics to enable real-time communication between modules.
